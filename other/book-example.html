Responsive Behavior

The book layout automatically scales and adapts to different screen sizes:

  • Desktop (1024px+): Full spread with 85% scale
  • Tablet (768px): 65% scale, spine hidden, vertical layout
  • Mobile (480px): 55% scale, optimized typography
/* Responsive scaling in CSS */ @media (max-width: 1024px) { .book-spread { --spread-scale: 0.85; } } @media (max-width: 768px) { .book-spread { --spread-scale: 0.65; width: 100vw; height: auto; } .book-spread::after { display: none; } /* Hide spine */ } @media (max-width: 480px) { .book-spread { --spread-scale: 0.55; } .title { font-size: 28px; } .subtitle { font-size: 20px; } .body { font-size: 16px; } }
