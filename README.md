# cleann

A local web-based flipbook and page manager project.

## Overview

This repository contains HTML, CSS, JavaScript, and assets for a flipbook-style web interface. It appears to include examples, layouts, and a page manager.

## Files

- `index.html` and related HTML pages for the flipbook demo
- `flipbook.css`, `style.css`, and other CSS files for layout and styling
- `book-groups.js` and `script.js` for app behavior
- `package.json` for Node package metadata
- `node_modules/` for installed dependencies

## Setup

Install dependencies if you want to work with the Node package metadata:

```bash
npm install
```

## Usage

Open the HTML files in a browser to preview the project, for example:

- `flipbook-test.html`
- `other/flipbook-book.html`
- `page-manager.html`
